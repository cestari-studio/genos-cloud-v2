🔄 Pipelines & Workflows

Pipeline: Geração de Post (Content Factory)

Factory.tsx → "Gerar Post" → api.edgeFn('content-factory-ai', body) → Edge: busca brand_dna + tenant_config → Monta prompt com char_limits + hashtag_strategy + content_rules → Gemini 2.0/2.5 API → Parse JSON response → Insert content_items → debit_credits() RPC → Insert activity_log → MatrixList atualiza

Pipeline: Autenticação (Wix Bridge)

MasterLogin → email → wix-auth-bridge Edge Function → Busca tenant por email → Auto-provision se novo → Shadow auth user (admin API) → Return session + JWT → AuthContext.login()

Pipeline: Notificações (Polling)

Shell.tsx mount → setInterval(30s) → SELECT activity_log (50 items) + SELECT popup_events (30 items) → Dedup (2s window) → Sort by created_at DESC → Bell badge + panel

Pipeline: Brand DNA → Geração

BrandDna.tsx editor → UPDATE brand_dna (voice, persona, rules, char_limits, hashtags, pillars) → content-factory-ai lê brand_dna → Inclui no system prompt: char_limits, fixed_footer, fixed_hashtags, categories → Gemini gera respeitando DNA

Pipeline: Settings → Configuração

Settings.tsx → SELECT tenant_config WHERE tenant_id = child → Edit: tokens, posts, formats, AI model, char limits, billing, contact → UPSERT tenant_config + UPDATE credit_wallets → Salvo

🪙 Tokens, Créditos & Billing

Fluxo de Tokens

tenant_config.token_balance → credit_wallets.prepaid_credits → debit_credits() após geração → usage_logs registra custo → Shell.tsx exibe saldo

Exibição: Shell.tsx calcula used / (used + remaining) do mês corrente.
PageLayout.tsx: Badge inline ■ AI {credits} / {maxCredits} tokens no header.
Atualização: Uma vez por mount do Shell (sem polling de tokens).
