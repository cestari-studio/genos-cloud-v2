Carbon AI Chat

Planejamento de integração do @carbon/ai-chat v1.7.0 no genOS Cloud Platform

Em Planejamento

Componentes Disponíveis

AiChat

Container principal para integração da interface de chat

Uso: Wrapper raiz para a funcionalidade de IA no genOS

MessageList

Exibe a lista de mensagens trocadas na conversa

Uso: Histórico e contexto da conversa do usuário

UserMessage

Bolha de mensagem do usuário

Uso: Renderização de inputs do usuário no chat

AssistantMessage

Bolha de resposta da IA

Uso: Renderização de respostas do assistente IA

PromptInput

Campo de texto + botão enviar

Uso: Interface de entrada de prompts no chat

MessageActions

Botões de ação: copiar, regenerar, feedback

Uso: Controles de interação com mensagens

AiChatHeader

Barra de cabeçalho do chat

Uso: Título e controles da janela de chat

Casos de Uso no genOS

Assistente de Conteúdo

IA para ajudar na criação e otimização de posts e conteúdo

Integração Meta

Chat para orientação e troubleshooting de conexões Meta

Diagnóstico de Bugs

IA para análise e sugestão de soluções para problemas técnicos

Geração de Prompts

Assistente para criar e refinar prompts de geração de código

Analytics Q&A

Interface de perguntas sobre dados de analytics e performance

Requisitos Técnicos

Dependência: @carbon/ai-chat ^1.7.0
React: 18.3 compatibility
Theme: g100 theme integration
Recursos: Streaming response support
Backends: OpenAI, Anthropic, Gemini

Prompt de Implementação

Mega Prompt para Implementação 
Você é um especialista em React e Carbon Design System. Implemente a integração completa do @carbon/ai-chat v1.7.0 no genOS Cloud Platform com os seguintes requisitos:

1. ESTRUTURA DE COMPONENTES
   - Crie um componente AiChatWidget que envolve toda a funcionalidade
   - Integre MessageList, UserMessage, AssistantMessage, PromptInput
   - Implemente MessageActions (copiar, regenerar, feedback)
   - Use AiChatHeader para o cabeçalho

2. FUNCIONALIDADES
   - Suporte a streaming de respostas da IA
   - Histórico de mensagens persistido em Zustand store
   - Integração com múltiplos provedores (OpenAI, Anthropic, Gemini)
   - Ações de mensagens: copiar conteúdo, regenerar resposta, enviar feedback

3. DESIGN E STYLING
   - Aplicar tema g100 (dark mode) do Carbon Design System
   - Componentes responsivos
   - Animações suaves para entrada/saída de mensagens
   - Indicador de carregamento durante streaming

4. CASOS DE USO
   - Assistente de criação de posts (/create)
   - Helper para integração Meta (/social)
   - Chat de diagnóstico de bugs
   - Gerador de prompts para IA
   - Interface Q&A para analytics (/analytics)

5. ESTADO E DADOS
   - Use Zustand para gerenciar state global do chat
   - Use TanStack Query para chamadas à API
   - Implementar cache de respostas
   - Persistir histórico em localStorage

6. TYPESCRIPT
   - Tipagem forte para todas as props e states
   - Interfaces para Message, ChatSession, Provider
   - Tipos para diferentes tipos de respostas

7. TESTES
   - Testes unitários com Vitest
   - Testes de integração para fluxos de chat
   - Testes de acessibilidade com a11y

Implemente com melhorias de performance e UX de acordo com as melhores práticas do Carbon Design System.
